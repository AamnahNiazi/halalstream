🟢 HalalStream

“Indeed, with hardship comes ease.” (Qur’an 94:6)

HalalStream is a modern Islamic streaming platform built to provide a calm, distraction-free environment for consuming beneficial Islamic content. It removes the noise of algorithms, toxic recommendations, and comment sections—offering a more intentional and spiritually focused viewing experience.

✨ Overview

In today’s digital world, content platforms are often designed to maximize engagement rather than peace of mind. HalalStream was created as an alternative—a space where users can access Islamic knowledge without distractions.

From Quran recitations to Seerah, nasheeds, naats, and lectures, everything is curated and filtered to maintain a clean, respectful, and focused experience.

“The most beloved deeds to Allah are those that are consistent, even if they are small.”
— Sahih al-Bukhari

📸 Preview



Clone the repository:

git clone https://github.com/your-username/halalstream.git
cd halalstream

Install dependencies:

npm install
# or
yarn install
🔐 Environment Variables

Create a .env.local file in the root directory:

NEXT_PUBLIC_YOUTUBE_API_KEY=your_api_key_here
▶️ Running the Project

Start the development server:

npm run dev

Then open:

http://localhost:3000
